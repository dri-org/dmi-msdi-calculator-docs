# MSDI Calculator Documentation

Supporting methodologies, technical memoranda, validation documentation, and release notes for the **MSDI Calculator** (Manure Subsurface Drip Irrigation Economic Feasibility Calculator).

---

## Overview

The MSDI Calculator is a screening-level decision-support tool developed to assist dairy producers, consultants, engineers, conservation professionals, researchers, and technical reviewers in evaluating the potential economic feasibility of implementing manure subsurface drip irrigation (mSDI-e).

The application combines user-provided information with county-specific climate data, crop and soil characteristics, manure management information, engineering relationships, published technical coefficients, and screening-level assumptions to estimate potential changes in:

- Capital investment
- Operating expenses
- Irrigation water use
- Nutrient utilization
- Potential environmental co-benefits

This repository contains the technical documentation supporting the current implementation of the application.
